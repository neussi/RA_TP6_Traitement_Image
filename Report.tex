\section{Introduction}

Ce rapport presente le module \textbf{Traitement d'Image from Scratch} du cours
\textit{Realite Augmentee FROM SCRATCH}. En AR, le traitement d'image est la porte
d'entree de tout le pipeline : detection de marqueurs, extraction de features, segmentation.

\textbf{Contrainte absolue :} zero bibliotheque tierce (pas d'OpenCV, pas de stb\_image).
Uniquement C++17 et STL.

\section{Concepts Theoriques}

\subsection{Structure d'image NkImage}

L'image est representee comme un tableau de pixels RGB en memoire contigue :
\[
    \text{pixel}(x, y) = \text{data}[(y \times w + x) \times 3]
\]

\begin{lstlisting}[language=C++, caption=Structure NkImage]
struct NkImage {
    int width, height;
    std::vector<uint8_t> data; // RGB entrelacess
    NkImage(int w, int h) : width(w), height(h), data(w*h*3, 0) {}
    void SetPixel(int x, int y, uint8_t r, uint8_t g, uint8_t b);
    void GetPixel(int x, int y, uint8_t& r, uint8_t& g, uint8_t& b) const;
    bool SavePPM(const std::string& path) const;
};
\end{lstlisting}

\subsection{Conversion en Niveaux de Gris}

Formule standard (ponderation perceptuelle) :
\[
    I_{gray} = 0.299 \cdot R + 0.587 \cdot G + 0.114 \cdot B
\]

\subsection{Image Integrale}

L'image integrale permet de calculer la somme de n'importe quelle region rectangulaire en $O(1)$ :
\[
    S(x, y) = \sum_{i \leq x, j \leq y} I(i, j)
\]
Construction en $O(mn)$, puis acces en $O(1)$ :
\[
    \text{Somme}(r) = S[x_2][y_2] - S[x_1-1][y_2] - S[x_2][y_1-1] + S[x_1-1][y_1-1]
\]

\subsection{Seuillage Adaptatif (Sauvola)}

Le seuillage adaptatif calcule un seuil local par fenetre :
\[
    T(x,y) = \mu(x,y) \cdot \left(1 + k\left(\frac{\sigma(x,y)}{R} - 1\right)\right)
\]
avec $\mu$ : moyenne locale, $\sigma$ : ecart-type local, $k=0.5$, $R=128$.

\subsection{Trace de Segments (Bresenham)}

\begin{lstlisting}[language=C++, caption=Algorithme de Bresenham]
void DrawLine(int x0, int y0, int x1, int y1, uint8_t r=255) {
    int dx = std::abs(x1-x0), dy = std::abs(y1-y0);
    int sx = (x0<x1)?1:-1, sy = (y0<y1)?1:-1;
    int err = dx - dy;
    while (true) {
        SetPixel(x0, y0, r, r, r);
        if (x0==x1 && y0==y1) break;
        int e2 = 2*err;
        if (e2 > -dy) { err -= dy; x0 += sx; }
        if (e2 <  dx) { err += dx; y0 += sy; }
    }
}
\end{lstlisting}
